interpolate

Type: array
Default: []
Calculates intermediate values between active and non-active pages during swipe.
Normally, when swiping, the visible pages will gain their respective focused and unfocused classes once a scroll occurs. For carousels where focused and unfocused pages are visually identical, this poses no problem. However, in cases where focused pages appear different, this setting will help to make the transition fluid.

Note that this only works for numerical CSS values. Additionally, be mindful of the scope and number of interpolations that are happening. Having many complex values changing can have performance issues on less powerful devices, since they are calculated, applied, and rendered at every instance of a swipe movement.

Each item in the array is an object containing each of the following values:
  • between
  • value
  • unit
The between option is an array that must contain two arrays, each of which correspond to a page position. Both of these sub-arrays must contain two values: the first value in the array selects the position of a page to be interpolated from, and the second value defines the numerical value that the page will have when perfectly at that position. The second sub-array is the same, but for the other page to be interpolated to. Note that these two pages MUST be directly adjacent to each other.

For example, to tell Roundabout that we want a page at position 0 to have a height of 80%, and a page at position at position 1 to have a height of 100%, our between value would look like this:
between: [
  [0, 80],
  [1, 100]
],
The value option is a string that selects the CSS value to manipulate. This value must be in camelCase, not kebab-case. Continuing our example, the value option will look like this:
value: "height",
The unit option defines the appropriate units to use for the CSS value. The string must contain a dollar sign ($) to mark the position that the calculated numerical value should be placed. To use percent for height, it would look like this:
unit: "$%"
All together, our interpolate setting will look like this:
interpolate: [
  {
    between: [
      [0, 80],
      [1, 100]
    ],
    value: "height",
    unit: "$%"
  }
]

Multiple interpolations can be applied to the same pairs of pages, but each requires its own object. For example, we can combine a brightness filter and height with these settings:
interpolate: [
  {
    between: [
      [0, 30],
      [1, 100]
    ],
    value: "filter",
    unit: "brightness($%)"
  },
  {
    between: [
      [1, 100],
      [2, 30]
    ],
    value: "filter",
    unit: "brightness($%)"
  },
  {
    between: [
      [0, 80],
      [1, 100]
    ],
    value: "height",
    unit: "$%"
  },
  {
    between: [
      [1, 100],
      [2, 80]
    ],
    value: "height",
    unit: "$%"
  },
]
